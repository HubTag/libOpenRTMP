title: rtmp_amf_version_t
--------------------------

Provides some symbols to specify AMF version.


## Syntax ##

```c
#include<openrtmp/rtmp/rtmp_constants.h>

typedef enum {
    RTMP_AMF0 = 0,
    RTMP_AMF3 = 3
} rtmp_amf_version_t;
```

## Members ##

#### RTMP_AMF0 ####
> Indicates AMF version 0
> 

#### RTMP_AMF3 ####
> Indicates AMF version 3
>
